# Sangmesh Client Portfolio

A React + Vite portfolio site for video and reel-based work, with animated project cards, a fullscreen media modal, and a contact form powered by Web3Forms.

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

- `npm run dev` - start the Vite dev server
- `npm run build` - create a production build
- `npm run preview` - preview the production build locally
- `npm run lint` - run ESLint

## Notes

- Video assets are stored in `public/` and referenced by absolute paths.
- The contact form submits to Web3Forms using the access key in `src/App.jsx`.
- Update the form key or move it to an environment variable before production use.
